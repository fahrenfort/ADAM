DESCRIPTION 

y=nt_mmat(x,m) -  matrix multiplication (with convolution)

  y: result
 
  x: input data (2D or more)
  m: matrix to apply (2D: right multiply, 3D: same with convolution)

  If m is 3D, the last index (k) is lag.
